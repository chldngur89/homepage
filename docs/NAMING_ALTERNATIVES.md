# ZeroSeller 대안 브랜드 네이밍

**포지션**: CMO AI Agent · Zero Click 강점 · "사장님은 대시보드만 보면 된다"

---

## 후보 이름 (의미 중심)

| 이름 | 의미·연상 | 한 줄 |
|------|-----------|--------|
| **DashCMO** | Dashboard + CMO. 대시보드가 곧 CMO. | "대시보드만 보면 되는 CMO" |
| **OneLook** | 한눈에(One Look). 대시보드 한 번만 보면 됨. | "한눈에 보는 CMO AI" |
| **Glance CMO** | Glance = 한눈에. 사장님은 흘끗 보기만. | "흘끗 보기만 하면 되는 CMO" |
| **ZeroWork CMO** | Zero Click → Zero Work. 손댈 일 없음. | "일 안 하는(안 해도 되는) CMO" |
| **JustDash** | Just the Dashboard. 필요한 건 대시보드뿐. | "대시보드만 있으면 됨" |
| **OnlyBoard** | Board = 대시보드. 오직 보드만. | "보드만 보면 되는 에이전트" |
| **Decido** | Decide + do. 결정만 하면 실행은 AI. (스페인어 decide) | "결정만 하세요, 실행은 AI" |
| **Nod AI** | 끄덕이기(Nod)만 하면 승인. | "끄덕이기만 하면 되는 AI" |
| **BriefCMO** | 브리핑만 받으면 됨. CMO가 브리핑해 줌. | "브리핑만 받는 CMO" |
| **SingleView** | Single View. 하나의 화면(대시보드)으로 충분. | "한 화면으로 끝나는 CMO" |
| **HandsOff CMO** | Hands-off. 손 댈 필요 없음. | "손 대지 않는 CMO" |
| **AutoCMO** | 자동 CMO. Zero Click과 잘 맞음. | "자동으로 돌아가는 CMO" |
| **Chief Desk** | 대표 책상. 책상 위 대시보드만 보면 됨. | "대표님 책상 위 CMO" |
| **View Only** | 보기만 하면 됨. (View Only) | "보기만 하면 되는 AI CMO" |
| **ZeroInput** | 입력 제로. Zero Click과 동일 맥락. | "입력 없이 돌아가는 CMO" |

---

## 추천 3개 (짧은 이유)

1. **DashCMO** — 대시보드 + CMO가 바로 떠오르고, "대시보드만 보면 된다"가 이름에 담김.
2. **OneLook** — 짧고 기억하기 쉬우며, "한눈에 / 한 번만 보면 된다"가 직관적.
3. **JustDash** — "Just the Dashboard" 느낌으로, 대시보드 중심 포지셔닝이 분명함.

---

## 유지 시 (Zero 계열만 다듬기)

- **ZeroSeller** → **ZeroClick CMO** (Zero Click + CMO를 이름에)
- **ZeroSeller** → **ZeroTouch** (터치/클릭 제로)
- **ZeroSeller** → **ZeroInput CMO** (입력 제로 CMO)

---

정한 이름이 있으면 `src/content/site.json`의 `siteName`과 각 페이지 노출처만 바꿔 주면 됩니다.
